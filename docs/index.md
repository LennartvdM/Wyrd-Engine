# Wyrd Engine Documentation

Welcome to the Wyrd Engine documentation hub. Start here to understand the project goals, explore its simulation components, and find links to deeper technical references.

## Overview

Wyrd Engine experiments with procedural world-building tools that generate believable daily routines, economic rhythms, and other systemic narratives for fictional settings. Each component favours iterative design: we release a working version, capture lessons from production use, and refine the architecture without discarding the valuable pieces.

For quick setup instructions and deployment tips, see the [usage guide](./usage.md).
